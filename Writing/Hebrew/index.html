    Consonants¶

    Explanation: when a consonant contains an inner dot, it is mostly a dagesh forte.

    When the BGDKPT letters contain a dot, it is either dagesh forte or a dagesh lene.

    When the H contains a dot, it is called a mappiq.

    transcription glyph phonetic remarks name unicode
    > א ʔ when not mater lectionis letter alef 05D0
    B ב bb / b / v forte / lene / normal letter bet 05D1
