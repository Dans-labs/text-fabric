# Combine

## Usage

```python
from tf.compose import combine
combine(
  (
      location1,
      location2,
  ),
  targetLocation,
)
```

```
combine(
  (
      (name1, location1),
      (name2, location2),
  ),
  targetLocation,
  componentType=None,
  componentFeature=None,
  featureMeta=None,
  **otext,
)
```
